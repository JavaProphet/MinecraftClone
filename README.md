# MinecraftClone
A MC clone in C, protocol-compatible with 1.9.

Dependencies (Linux): 
libpng

libz

libglfw3 (statically linked)

libX11

libdl

libXrandr

libXinerama

libXi

libXxf86vm

libXcursor

libpthread

libm

libGL

libGLU



Dependencies (Windows): 

libz

glew32

glfw3 (statically linked)

opengl32

glu32

gdi32

ws2_32

libpng
